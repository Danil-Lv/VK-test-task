# VK-test-task
### Запуск проекта

Клонировать репозиторий:

```
git clone https://github.com/Danil-Lv/shortlink.git
```

Установить зависимости
```
pip install req.txt
```

Запустить сервер 
```
python3 manage.py runserver
```
Документация API будет доступна по ссылке 
`http://127.0.0.1:8000/swagger/`
